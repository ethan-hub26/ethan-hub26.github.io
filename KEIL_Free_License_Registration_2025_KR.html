절차 요약

Keil uVision 실행 → File > License Management → User-Based License 탭 이동 → Activate/Deactivate → License Server 선택 → 서버 주소 입력 후 Query 와 Activate 순서로 진행합니다.

1. License Management 열기
  1. Keil uVision을 실행합니다.
  2. 상단 메뉴에서 File 을 클릭합니다.
  3. License Management 메뉴를 선택합니다.
Figure 1. File > License Management 진입 화면
Figure 2. User-Based License 탭과 Activate/Deactivate 버튼
2. User-Based License 탭으로 이동
  1. License Management 창에서 상단 탭을 확인합니다.
  2. User-Based License 탭으로 이동합니다.
  3. Activate/Deactivate 버튼을 눌러 등록 절차를 시작합니다.

이 단계는 PDF 3페이지의 핵심 내용입니다.

기존 장치 기반 등록이 아니라, Nuvoton Edition의 무료 사용자 기반 라이센스를 등록하는 흐름으로 이해하면 됩니다.

3. License Server 등록
  1. License Server 항목을 선택합니다.
  2. 서버 주소에 아래 URL을 입력합니다.
  3. Query 버튼을 눌러 서버 응답을 확인합니다.
  4. Activate 버튼을 눌러 라이센스를 등록합니다.
  5. Keil uVision을 다시 시작한 뒤 사용합니다.

2PPTX 연계 섹션

아래 내용은 Nuvoton_개발환경구축_통합가이드.pptx 의 Part 1에서, 무료 라이센스 등록과 직접 연결되는 섹션만 추려서 줄 단위로 재구성한 것입니다.

2.1 개요 및 준비 사항
  • 대상 MCU는 Nuvoton Cortex-M Series입니다.
  • IDE는 Keil MDK-ARM이며, M0 / M23 대상 무료 라이센스를 제공합니다.
  • 디버거는 Nu-Link 또는 Nu-Link-Me를 사용합니다.
  • 개발환경은 Keil MDK + Nu-Link + BSP 조합입니다.
  • 양산환경은 ICP Tool / ISP Tool 중심으로 구성합니다.
  • 준비물은 Windows PC, SDK 보드, 인터넷 연결, Nu-Link 디버거입니다.
2.2 설치 절차 전체 흐름

설치 순서

  • Keil 다운로드
  • 설치 실행
  • 라이센스 취득
  • 라이센스 등록
  • SW Pack 설치

주의 사항

  • 기본 설치 경로 사용을 권장합니다.
  • 한글/공백 경로는 가급적 피합니다.
  • Nu-Link 드라이버 설치 전에는 Keil을 종료합니다.
  • 드라이버와 펌웨어 버전이 다르면 업데이트를 권장합니다.
2.3 BSP 다운로드 및 컴파일 테스트
  • Nuvoton 홈페이지의 BSP and Sample Code에서 패키지를 받을 수 있습니다.
  • 최신 BSP는 OpenNuvoton GitHub 에서 MCU 모델명으로 찾을 수 있습니다.
  • ZIP 압축 해제 후 Sample 프로젝트의 .uvproj 파일을 열어 컴파일과 다운로드를 확인합니다.
2.4 유용한 도구

Nu-Link / ICP / ISP

  • 다운로드, 디버그, VCOM 기능을 제공합니다.
  • 배치 다운로드와 Chip Options 설정에 사용합니다.
  • 부트로더 기반 ISP 다운로드에도 활용합니다.

Pin / Clock Configure

  • GUI에서 핀 설정을 구성합니다.
  • C 코드 자동 생성이 가능합니다.
  • ClockConfig 모듈과 연동해 초기화 코드를 빠르게 만듭니다.
3FAQ 정리
  • 인터넷이 없는 PC는 해당 PC의 CID를 복사해 인터넷 가능한 PC에서 라이센스 코드를 취득한 뒤 다시 입력합니다.
  • SW Pack 설치가 안 되면 Legacy Support 패키지를 받아 수동 설치합니다.
  • Device not found 오류가 뜨면 Pack / Device 설정과 프로젝트 대상 칩을 다시 확인합니다.
  • Nu-Link 펌웨어 버전 불일치 경고가 뜨면 업데이트 후 USB를 다시 연결합니다.
  • 다운로드 실패 시 Debug / Utilities 탭의 Reset 설정을 점검합니다.
  • 리셋 없이 디버그에 진입하려면 Debug와 Utilities 탭에서 Reset 옵션을 해제합니다.
