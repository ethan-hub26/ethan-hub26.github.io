1 Introduction

Nuvoton의 Nu-Link 디버그 어댑터는 SWD(Serial NuMicro® Wire Debug) 신호 인터페이스를 기반으로 하는 USB 디버거 및 프로그래머이며 Nuvoton 제품군 칩 개발에 적용할 수 있습니다. 표 2-1에서 볼 수 있듯이 Nu-Link-Pro, Nu-Link, Nu-Link-Me 및 Nu-Link2-Me를 ​​포함하여 다양한 사양에 따라 Nu-Link 디버그 어댑터에는 4가지 유형이 있습니다. 특별한 조건이 언급되지 않는 한 일반적으로 4가지 유형을 "Nu-Link 어댑터"라고 합니다.

Nu-Link 어댑터는 SWD(직렬 와이어 디버그) NuMicro® 신호 인터페이스를 기반으로 하는 ICP(회로 내 프로그래밍)를 지원합니다. 사용자는 대량 생산을 위해 칩 펌웨어를 업데이트하기 위해 ICP 프로그래밍 도구를 사용할 수 있습니다. Nu-Link 어댑터는 Keil MDK, IAR EWARM 및 NuEclipse GCC와 같은 타사 개발 도구도 지원합니다.

단순성과 명확성을 위해 본 사용 설명서의 특정 용어의 일부는 다음 표에 나열된 대로 축약되거나 축약되었습니다.

짧은 이름 Nu-Link 어댑터 Nuvoton Nu-Link 디버그 어댑터 NuMicro® NuMicro® 제품군 Nuvoton 제품군 NuMicro® ICP 도구 Nuvoton ICP 프로그래밍 도구 Keil MDK Keil ARM RealView 마이크로컨트롤러 개발 키트(MDK-ARM®) IAR EWARM IAR ARM용 임베디드 워크벤치 NuEclipse GCC NuEclipse 통합 개발 환경 SWD 직렬 와이어 디버그 ICP 회로 내 프로그래밍

2 Hardware Specifications

Nu-Link 어댑터는 대상 칩에 연결하기 위한 USB 커넥터와 SWD 신호 인터페이스를 제공합니다. 사용자는 Nu-Link 어댑터를 PC의 USB 포트에 연결하여 개발 소프트웨어 도구를 통해 대상 칩을 디버깅하고 프로그래밍할 수 있습니다. 표 2-1과 같이 Nu-Link 어댑터에는 3가지 사양이 있으며, 사양에 따라 디버깅, 온라인/오프라인 프로그래밍, SWD I/O 전압 설정이 지원될 수 있습니다. (자세한 내용은 +부록 참조)

*1 저항 JPR1로 조정됩니다.

*2 저항 ICEJPR1로 조정됩니다.

*3 입력은 1.8V~5V를 지원하고, 출력은 1.8V~3.3V만 지원합니다.

*4 Nu-Link2-Me는 제어 버스를 통해 자동 IC 프로그래밍 시스템에 연결할 수 있습니다.

IC 프로그래머를 위한 제어 버스(Start, Busy, Pass, Fail) .

*5 Virtual COM은 V3.0 이후 버전에서 지원됩니다.

2.1 Nu-Link-Pro

Nu-Link-Pro는 디버깅, 온라인/오프라인 프로그래밍, SWD I/O 전압 설정 기능을 갖춘 완전한 기능의 디버거이자 프로그래머입니다. 그림 2-1에 표시된 것처럼 Nu-Link-Pro에는 컴퓨터 호스트에 연결할 수 있는 USB 포트, 상태 LED 세트, 오프라인 프로그래밍 버튼, 디버깅 및 프로그래밍을 위해 대상 칩에 연결할 수 있는 SWD 포트(SWD 포트의 전압 레벨은 소프트웨어를 통해 1.8V, 2.5V, 3.3V 또는 5.0V로 조정 가능), SWD I/O 전압 LED 및 SWD 전원 출력 LED 세트가 포함되어 있습니다.

Offline Programming Button SWD Power Output LED SWD VCC NC ICE_DAT NC USB ICE_CLK NC /RESET NC VSS(GND) NC SWD I/O Voltage LED Status LED (1.8V / 2.5V / 3.3V / 5.0V) (ICE / ICP / Red / Green)

2.2 Nu-Link

Nu-Link는 디버깅 및 온라인/오프라인 프로그래밍 기능을 갖춘 기본 디버거이자 프로그래머입니다. 그림 2-2에서 볼 수 있듯이 Nu-Link에는 컴퓨터 호스트에 연결할 수 있는 USB 포트, 상태 LED 세트, 오프라인 프로그래밍 버튼, 디버깅 및 프로그래밍을 위해 대상 칩에 연결할 수 있는 SWD 포트가 포함되어 있습니다(SWD 포트의 기본 전압은 5.0V).

Offline Programming Button SWD USB Green Status LED Red ICP ICE

2.3 Nu-Link-Me

Nu-Link-Me는 디버깅 및 온라인 프로그래밍 기능을 갖춘 간단한 디버거이자 프로그래머로, NuTiny-SDK 키트와 함께 제공되며 NuMicro® 맞춤형 제품군 시스템을 개발하기 위해 독립형으로 사용할 수 있습니다. 그림 2-3에 표시된 것처럼 Nu-Link-Me에는 컴퓨터 호스트에 연결할 수 있는 USB 포트, 상태 LED 세트, Nu-Link-Me의 전압을 3.3V와 5.0V(기본값은 3.3V) 사이에서 전환하는 전원 스위치, 디버깅 및 프로그래밍을 위해 대상 칩에 연결할 수 있는 SWD 포트(전압은 NuLink-Me로 조정 가능)가 포함되어 있습니다.

Virtual COM Switch SWD VCC ICE_DAT ICE_CLK USB /RESET ICE_RX VSS(GND) ICE_TX Status LED (ICE / ICP / Red / Green)

2.4 Nu-Link2-Me

Nu-Link2-Me는 디버깅 및 온라인 프로그래밍 기능을 갖춘 간단한 디버거 및 프로그래머로, Nu-Maker 키트에만 제공되며 SWD 인터페이스를 통해 온라인 프로그래밍 및 디버깅을 지원하는 맞춤형 제품군 시스템인 NuMicro® 개발을 위해 독립형으로 사용할 수 있습니다. 온보드 16Mbit SPI 플래시를 사용하면 대상 마이크로컨트롤러를 오프라인으로 프로그래밍할 수 있습니다. Nu-Link2-Me는 PC에서 메시지를 인쇄할 수 있는 가상 COM 포트(VCOM) 기능을 제공합니다. Nu-Link2-Me는 NuMaker와 분리되어 사용자가 대량 생산 프로그래밍 도구로 사용할 수 있습니다. 그림 2-4는 주요 구성요소와 커넥터를 전면에서 본 모습입니다.

3 Main Functions

NuMicro® Nu-Link 어댑터는 제품군을 위한 완벽한 디버깅 및 프로그래밍 기능을 제공하고 다양한 타사 개발 도구를 지원합니다. 자세한 기능 지원은 표 3-1에 나열되어 있습니다.

*1 코어 레지스터 보기가 지원됩니다. 주변기기 보기는 지원되지 않습니다.

*2 Nu-Link, Nu-Link-Pro 및 Nu-Link2-Me에 지원됩니다.

*3 Nu-Link-Pro에 지원됩니다.

3.1 Debugging

이 섹션에서는 Nu-Link Adapter가 지원하는 디버깅 기능에 대해 간략하게 설명합니다. 자세한 내용은 관련 사용자 매뉴얼을 참조하시기 바랍니다.

3.1.1 Debug Mode

NuMicro® Nu-Link 어댑터는 SWD 신호 인터페이스를 기반으로 하는 제품군 칩에 대한 디버깅을 지원합니다. 칩 디버깅을 위해 Nu-Link 어댑터 사용을 지원하는 타사 도구에는 Keil MDK, IAR EWARM 및 NuEclipse GCC가 포함됩니다. 디버그 모드에서 지원되는 추가 기능은 다음과 같습니다.

3.1.2 Breakpoints

디버그 모드에서 사용자는 디버깅을 위해 코드에 중단점을 추가할 수 있습니다. Nu-Link 어댑터의 실시간 시뮬레이션 중에 칩 시뮬레이션은 특정 중단점에서 중지됩니다. 그림 3-1은 Keil MDK 디버그 모드의 중단점 설정을 보여줍니다. 052행과 059행의 빨간색 레이블은 삽입된 중단점을 나타냅니다. 노란색 화살표는 다음에 실행될 코드를 나타내며 프로그램 카운터(PC)의 레지스터 값을 표시합니다(예: 그림 3-1의 "R15(PC)=0x00000D04" 레지스터 창).

3.1.3 Direct Register Control Interface

직접 레지스터 제어 인터페이스는 대상 칩에 레지스터 내용을 표시하고 레지스터를 조작하는 데 사용할 수 있습니다. 예를 들어 Keil MDK 디버그 모드를 선택하면 디버그 명령을 호출하고(예: ADC, CAN, CLK 등) 직접 레지스터를 열고 그림 3-2와 같이 선택한 레지스터의 "기능 레지스터 목록" 제어 인터페이스에서 레지스터를 선택합니다.

CLK용 직접 레지스터 제어 인터페이스는 그림 3-3의 왼쪽 부분에 표시됩니다. 여기서 왼쪽 열은 레지스터 주소를 표시하고, 중간 열은 레지스터 이름을 표시하고, 오른쪽 열은 레지스터 값을 표시합니다. PWRCON용 직접 레지스터 제어 인터페이스는 그림 3-3의 오른쪽 부분에 표시되어 있으며, 왼쪽 열에는 기능 비트가 표시되고 중간 열에는 기능 이름이 표시되며 오른쪽 열에는 기능 값이 표시됩니다.

Detailed Operation:

"레지스터 값"을 두 번 클릭하면 그림 3-3의 오른쪽 부분에 표시된 것처럼 레지스터 제어 세부 정보가 열립니다.

"레지스터 이름" 또는 "컨트롤 값" 위로 커서를 이동하면 팁이 표시됩니다. "레지스터 값" 또는 "제어 값"은 직접 수정할 수 있습니다. 그러면 Nu-Link 어댑터가 대상 칩의 내용을 수정합니다.

3.1.4 Semihost

NuMicro® Semihost 기능을 사용할 때 제품군 마이크로 컨트롤러의 메시지는 Nu-Link 어댑터를 통해 UART를 통해 디버그 창으로 출력될 수 있습니다. 즉, GPIO 없이 메시지가 출력됩니다. 그림 3-4는 Nu-Link Adapter에서 출력되는 메시지인 “UART #1” 형식의 디버그 메시지를 보여줍니다.

Semihost를 사용하려면 아래 단계를 따르십시오(Keil MDK 및 M031AE 시리즈를 예로 사용).

1단계: "startup_NUC1xx.s" 문자열을 다음과 같이 수정합니다.

    ;SEMIHOSTED SETL {FALSE} ; 이 줄 삭제

    ;↓수정

    SEMIHOSTED SETL {TRUE} ; 이 줄을 추가하세요

2단계: "system_NUC1xx.h" 문자열을 다음과 같이 수정합니다.

    //#define DEBUG_ENABLE_SEMIHOST // 이 줄을 삭제합니다.

    //↓수정

    #define DEBUG_ENABLE_SEMIHOST // 이 줄을 추가하세요

3단계: Rebuild를 호출하여 프로젝트를 다시 빌드하고 디버그 모드로 들어갑니다.

4단계: 디버그 모드에서 그림 3-4와 같이 View → Serial Windows → UART #1을 호출합니다.

5단계: F5를 눌러 대상 칩을 프로그래밍하면 디버그 메시지가 UART #1 형식으로 출력됩니다.

3.2 Programming

이 섹션에서는 Nu-Link 어댑터가 지원하는 프로그래밍 기능을 간략하게 설명합니다. 자세한 내용은 관련 사용자 매뉴얼을 참조하시기 바랍니다.

3.2.1 Online Programming

온라인 프로그래밍은 그림 3-5와 같이 Nu-Link 어댑터가 소프트웨어 프로그램을 통해 제품군 단일 칩의 펌웨어를 대상 칩에 다운로드할 수 있음을 의미합니다.

3.2.2 Offline Programming

NuMicro® 오프라인 프로그래밍은 Nu-Link 어댑터가 소프트웨어 프로그램에 액세스하지 않고도 제품군 단일 칩의 펌웨어를 직접 업데이트할 수 있음을 의미합니다(그림 3-6 참조). 오프라인 프로그래밍은 대량 생산에 유용합니다. 대량 생산에는 원본 코드나 펌웨어 파일을 전달할 필요가 없고 Nu-Link 어댑터만 있으면 되기 때문입니다. 또한 Nu-Link 어댑터는 승인된 펌웨어 수를 효과적으로 제어할 수 있는 "제한된 오프라인 프로그래밍"을 지원합니다. 자세한 내용은 ICP 도구 사용자 매뉴얼을 참조하세요.

3.2.3 Software Serial Number (SN)

ICP 도구에서 제공하는 소프트웨어 일련 번호(SN) 기능을 사용하면 사용자는 온라인/오프라인 프로그래밍 중에 대상 칩에 대한 "Increase SN from" "Write address in flash" 필드에 값을 지정할 수 있습니다. 예를 들어 NUC140VE3CN 칩을 사용하면 사용자는 APROM, LDROM 및 데이터 플래시에 "증가된 일련 번호(SN)" 및 "쓰기 주소" 세트를 지정할 수 있으며 기록된 일련 번호(SN)는 자동으로 증가됩니다(그림 3-7 참조).

3.3 Wide Voltage Programming

Nu-Link-Pro는 개발 소프트웨어 도구가 SWD 포트 전압을 1.8V, 2.5V, 3.3V 또는 5.0V로 조정할 수 있는 광범위한 전압 프로그래밍 기능을 지원합니다. 그림 4-2에서 볼 수 있듯이 제어할 수 있는 핀에는 VCC, ICE_DAT, ICE_CLK, /RESET이 있습니다.

또한, 그림 2-1과 같이 Nu-Link-Pro는 SWD 포트 전압을 확인하기 위한 SWD I/O 전압 LED와 SWD 전원 출력 LED 세트를 제공합니다. LED 상태에 대한 자세한 내용은 표 2-3을 참조하십시오. Nu-Link-Me는 저항기 JPR1이 SWD 포트 전압을 3.3V 또는 5.0V로 조정할 수 있는 넓은 전압 프로그래밍 기능을 지원합니다. 그림 4-2에서 볼 수 있듯이 제어할 수 있는 핀에는 VCC, ICE_DAT, ICE_CLK, /RESET, ICE_RX 및 ICE_TX가 있습니다.

Nu-Link2-Me는 저항기 ICEJPR1이 SWD 포트 전압을 1.8V, 3.3V 또는 5.0V로 조정할 수 있는 광범위한 전압 프로그래밍 기능을 지원합니다. 그림 4-2에서 볼 수 있듯이 제어할 수 있는 핀에는 VCC, ICE_DAT, ICE_CLK, /RESET, ICE_RX/PASS, ICE_TX/Fail, BUSY 및 START가 있습니다.

3.4 Nu-Link 어댑터 드라이버 설치

Nu-Link 어댑터는 다양한 기능과 타사 소프트웨어 도구(예: Keil MDK 및 IAR EWARM)를 지원합니다. 소프트웨어 프로그램을 설치한 후에는 드라이버도 필요합니다. Keil MDK용 Nu-Link 어댑터 드라이버 및 IAR EWARM용 Nu-Link 어댑터 드라이버 링크를 사용하여 최신 버전을 설치할 수 있습니다. 소프트웨어 설정에 대한 자세한 내용은 섹션 4.2를 참조하십시오.

4 설치 및 설정

4.1 Nu-Link 어댑터에 연결하기

그림 4-1에서 볼 수 있듯이 Nu-Link 어댑터는 소프트웨어 도구가 USB를 통해 대상 칩을 디버깅하고 프로그래밍할 수 있는 USB와 SWD 인터페이스 사이의 브리지입니다. 사용자는 Nu-Link 어댑터를 PC의 USB 포트에 직접 연결하거나 USB 커넥터를 사용하여 연결할 수 있습니다.

Nu-Link 어댑터는 SWD 포트를 통해 대상 회로 기판에 전원(1.8V, 2.5V, 3.3V 또는 5.0V)을 공급할 수 있습니다. 최대값은 5V/500mA입니다. 자세한 사양은 표 2-1을 참조하세요.

SWD Connector:

5.2 Automatic IC Programming System

그림 5-1과 같이 개별 슬롯과 제어 버스를 통한 자동 IC 프로그래밍 시스템.

Operation sequence

5.2.1 및 파형

1. Nu-Link2-ME의 전원을 켭니다. START, BUSY, PASS, FAIL이 로직으로 설정됩니다.

2. 프로그래밍을 시작하려면 START를 TSTART에 대해 논리 0으로 설정해야 합니다.

3. 프로그래밍 시작. BUSY는 로직 0으로 설정되며 프로그래밍 중에 토글될 수 있습니다.

4. 프로그래밍이 끝나면 BUSY는 로직 1로 설정되고 PASS 또는 FAIL은 로직 0으로 설정됩니다.

  • BUSY가 로직 1로 설정되고, PASS가 로직 0으로 설정되면 “PASS”를 의미합니다.
  • BUSY가 로직 1로 설정되고, FAIL이 로직 0으로 설정되면 “FAIL”을 의미합니다.
