Folder structure

  • "Template" 또는 "RegBased" 디렉터리 내의 하위 폴더와 동일한 수준에 새 프로젝트 폴더를 생성하는 것이 좋습니다.
  • "라이브러리" 폴더와 모든 하위 폴더는 항상 유지되어야 합니다.

.c 파일은 다른 컴파일러에 대해 정의됩니다.

  • Different Coding Rule

__C51__은 KEIL PK51용입니다.

__ICC8051__은 IAR 8051 워크벤치용입니다.

__SDCC__는 NuEclipse용입니다.

NuEclipse 정의 기호 창

  • Simplify code

특정 컴파일러를 사용하기로 결정하면 다른 컴파일러 전체 그룹의 정의를 제거할 수 있습니다.

프로젝트 구축

Utilizing BSP - KEIL

기존 프로젝트에서 복사하는 것이 새 프로젝트를 만드는 것보다 더 유리한 경우가 많습니다.

케일의 경우

  • "Project_temp" 템플릿에서 폴더 복사
  • "project_temp.uvproj"의 이름을 "Rename.uvproj"로 바꿉니다.
  • KEIL에서 프로젝트 이름을 수정합니다.
  • 새 애플리케이션 .c 파일로 컴파일합니다. 출력 hex 및 bin 파일을 찾습니다.

KEIL PK51의 옵션

  • 다음 소개에서는 Nuvoton 8051 시리즈의 특별한 정의에 중점을 둡니다.
  • 모든 옵션에 대한 포괄적인 설명을 보려면 공식 Keil 문서를 참조하세요.

https://www.keil.com/support/man/docs/uv4cl/uv4cl_dg_options.htm

KEIL의 경우 - 장치

  • "Nuvoton 8051 Device"가 선택 가능한지 확인하세요.
  • 이 옵션을 사용할 수 없는 경우 Nuvoton Keil 드라이버 패키지를 설치하십시오.

KEIL - 타겟

  • 기기를 확인해 보세요. 일치하지 않는 경우 "장치" 페이지에서 올바른 장치를 선택하세요.
  • Memory Model:

XDATA를 사용하는 변수 기본값의 경우 큼

KEIL - 출력용

  • "실행 파일 이름"은 출력 .hex 파일 이름을 정의합니다.
  • Keil은 HEX 형식 파일을 독점적으로 생성하는 반면 BIN 파일은 Nuvoton 명령줄을 사용하여 생성됩니다. 다음 페이지에서 소개하는 명령줄입니다.

KEIL의 경우 - 사용자

  • 이 파일은 Keil 설정 폴더에 있습니다. 예: C:\Keil_v5\C51\NULink

。 기본적으로 사용되지 않은 위치는 0xFF로 채워집니다.

。 사용자는 매개변수를 포함할 수도 있습니다. -P 0x00은 사용되지 않는 위치가 0으로 채워짐을 의미합니다.

。 매개변수 <시작 주소> <데이터 길이>

Utilizing BSP - IAR

기존 프로젝트에서 복사하는 것이 새 프로젝트를 만드는 것보다 더 유리한 경우가 많습니다.

IAR의 경우

  • "Project_temp" 템플릿에서 폴더를 복제합니다.
  • 그런 다음 ".ewd", ".ewp" 및 ".eww" 파일 이름을 적절하게 변경합니다.
  • ".eww" 파일을 열고 새 이름 "Rename.ewp"를 반영하도록 프로젝트 <path>를 수정합니다.
  • 새 애플리케이션 .c 파일로 컴파일합니다. 출력 hex 및 bin 파일을 찾습니다.

IAR 워크벤치 옵션

  • 다음 소개에서는 Nuvoton 8051 시리즈의 특별한 정의에 중점을 둡니다.
  • 모든 옵션에 대한 포괄적인 설명을 보려면 공식 IAR 문서를 참조하세요.

https://wwwfiles.iar.com/8051/webic/doc/EW8051_MigrationGuide_v7.pdf

IAR의 경우 – 일반 옵션

  • 기기를 확인해 보세요.
  • Data Model:

XDATA를 사용하는 변수 기본값의 경우 큼

  • Stack / Heap
  • XDATA 값은 printf에 중요합니다.
  • DPTRs
  • 항상 1개의 DPTR을 사용하십시오.
  • Printf 포맷터용 자동

  • 16진수 형식 출력으로 빌드
  • Bin 형식은 nuvoton 명령줄을 사용합니다. 다음 페이지에서 소개합니다.
